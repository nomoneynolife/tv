# 自动获取酒店源 #

#### 感谢 ssili126 大佬开源 https://github.com/ssili126/tv/ ####

使用方法：
  
      https://raw.githubusercontent.com/nomoneynolife/tv/main/itvlist.txt
  
手动获取：  
  
  1、安装python  
  2、pip install selenium requests futures eventlet  
  3、安装chrome，下载对应版本的chromedriver  
  4、下载/运行itvall.py  
  5、运行完成后在当前目录下生成itvlist.txt文件
